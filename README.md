# scratchの改造
教育用のノード型プログラミングアプリである[scratch](https://scratch.mit.edu/projects/editor/?tutorial=getStarted)
scratchの拡張機能として`line bot`, `Chat gpt`, `voice vox`を扱える
## require
```
npm: 8.19.4
Node.js: v16.20.2
```
- demo
scratch上でchat gptを使用した line botの構築

## インストール
```
make install
```

## 使用法
starting Scratch server
```
cd scratch-gui
npm start
```
